This repository includes example assignments, activities, and other course documents that connect to the resources on the site. The files are maintained in a public Google Drive folder.

How to Copy a Document

Click on any item listed to view the related document on Google Drive. To save a copy that you can customize for your courses, login to your own Google account and then use the Make a Copy... command under the File menu. The system will save a copy of the file to your own Google Drive. Check the Google Help on how to Create, find, or download a file for additional information.
